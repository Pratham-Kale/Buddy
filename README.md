# 🤖 Buddy - Your Mental Health Companion

Buddy is a simple and friendly web app designed to support your mental health. It features a secure login system and a persistent chat where you can talk about your feelings.

## 🚀 How to Run

Running Buddy is very easy! Just follow these steps:

1. **Install Python**: Make sure you have Python installed on your computer.
2. **Run the Server**: Open your terminal (or Command Prompt) in this folder and type:
   ```bash
   python main.py
   ```
3. **Wait a Moment**: The script will automatically install all the necessary packages for you.
4. **Start Chatting**: Your web browser will automatically open to the Buddy dashboard. If it doesn't, just go to `http://localhost:8000` in your browser.

## ✨ Features

- **Personalized Chat**: Buddy listens to you and responds based on your mood (happy, sad, anxious, etc.).
- **Secure Accounts**: Create your own account to keep your chats private.
- **Persistent History**: Your chats are saved safely in a database, so you never lose your progress.
- **Simple & Beautiful**: A clean, modern interface designed to make you feel comfortable.

## 🛠️ Project Structure

- `main.py`: The heart of the app. It runs the server and handles the backend.
- `static/`: This folder contains all the frontend files (HTML, CSS, and JS).
- `buddy.db`: This is where your data is saved (created automatically).
- `requirements.txt`: A list of the tools Buddy needs to work.
